⚙️ How the Crusher Works

  1. Fetch latest Bitcoin block hash (64 hex chars)
  2. Sum all hex digit values (0–15 each)
  3. Apply: total % 16 → channel index
  4. Same block hash → same channel, worldwide
  5. New block every ~10 minutes = new spin

📡 Virtual SDR Channels

Each channel maps to a virtual signal type: from smooth jazz streams to military-style chatter simulations, alien electroacoustic soundscapes, and live Bitcoin transaction sonification. No hardware required — pure software-defined radio.

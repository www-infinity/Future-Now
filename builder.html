⚙️ How It Works

  1. Enter your website description
  2. The engine tries Ollama (local) → built-in GPU engine → Gemini (optional)
  3. AI writes complete, unique HTML line-by-line — watch the log!
  4. Your admin device ID is embedded as permanent page owner
  5. Preview opens in a new tab instantly
  6. Page is committed to GitHub and goes live on GitHub Pages
